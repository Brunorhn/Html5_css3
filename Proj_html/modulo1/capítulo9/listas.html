Trabalhando com Listas

ol (listas ordenadas)

Pode-se usar os parâmetros "Type" e "Start" dentro do "ol"

  1. lista 1
  2. lista 2
  3. lista 3
    • Neste ponto, irei criar uma sublista não ordenada usando "ul" dentro do "ol"
  4. lista 4

Listas não ordenadas

Posso usar o parâmetro type, assim como na "ol"

  • lista 1

dl é usado para Lista de Definição

Lista de definição é é similar a um dicionário, onde você vai usar o termo (dt) e sua definição (dd), conforme exemplo abaixo:

HTML
Linguagem de marcação para a criação de conteúdo
CSS
Linguagem de marcação para a criação de designer de um site ou plataforma
javaScript
Linguagem de programação para a crianção de interatividade de um site ou plataforma
