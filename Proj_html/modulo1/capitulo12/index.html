CSS in line (primeiro h1)

Neste capítulo, veremos as formas de utilizar css no htm

características do CSS in line

  1. O comando "style" é usado dentro da tag onde busca atribuir a estilização
  2. O código é aplicado somente na linhas que o css está sendo aplicado
  3. O código de estilo fica junto do código htlm, deixando o visual poluído
  4. Necessita de trabalho manual por elemento, para estilizar ou alterar estilo já aplicado
