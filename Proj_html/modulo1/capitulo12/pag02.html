CSS Interno

Neste método, o css é aplicado no Header, utilizando-se do conceito de seletores

Principais características

  1. deve-se utulizar a tag style no header
  2. deve-se referenciar o elementto a aplicar o estilo, utilizando-se de chaves após o nome do elemento

Principais desvantagens

  1. O Estilo é aplicando apenas ao documento (página) onde o código está inserido
