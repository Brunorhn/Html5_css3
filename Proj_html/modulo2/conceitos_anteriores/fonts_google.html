Unsando Google Fontes

Considerações iniciais

Nesta aula, iremos fazer o uso de fontes externas, utilizando o google fontes, com os seguintes passos:

  1. Acesse fonts.google.com e escolha a fonte
  2. Na lateral direita, escolha a opção selecionar
  3. Agora basta escolher a opção referente ao código em css
  4. Em seguida, basta adicionar o código copiado junto a cabeça do seu código html, dentro da tag
